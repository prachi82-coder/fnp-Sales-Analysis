# FNP Sales & Revenue Dashboard (Excel)

This project is an interactive Excel dashboard created to analyze sales and revenue data for FNP (Ferns N Petals).
The dashboard provides insights into overall sales performance, top-selling products, order trends, and customer purchasing patterns.

Key Features:
- Sales and Revenue Analysis
- Top Products Performance
- Order Trends Visualization
- Region-wise Sales Insights
- Interactive charts and pivot tables

Tools Used:
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning & Analysis

Purpose:
The goal of this project is to demonstrate data analysis and dashboard creation skills using Excel for business insights.
